Counting Things

Tool to count objects, items, points, habits.

Counting Things App has two modes of use: - Single List Mode: where you can perform counts in an accumulated way, ideal for specific cases like counting coins for example. - Daily List Mode: where the counters will be accumulated per day, in this mode the counters will be reset to each day, you will be able to consult the counters of the previous days and their respective values, ideal for count daily habits or events of a daily character.
